"Sistema de Gestión Integral para Tienda de Abarrotes"

Redacción del problema: Registro manual de productos y ventas que genera errores en el stock y dificultad para saber qué se ha vendido.

Solución propuesta: Sistema para registrar productos, controlar inventario y registrar ventas diarias.

El sistema permite:
- Registrar productos
- Gestionar el inventario
- Integrar proveedor, ventas, clientes y promociones
